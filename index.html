Game Rules

A random color box will be guessed.
You will have a set of color boxes displayed below it.
Use the color boxes options to pick a matching color to guess the color.
Each correct guess equals 5pts , if you miss - 5pts will be deducted from your score.
The random color is within the color boxes available. You will see an hint button below, you can use it at anytime to get an idea.

Your score is displayed at the top
You have limited time per round which you will see at the top.
A quit button at the button of the screen to quite before time but your score will not be saved.
Your can check the high score in the menu options to see the current score and if you beat the highest score
Each round last 2mins and color guesses is changed every 2 secs,,,, HOW FAST CAN YOU BE???
